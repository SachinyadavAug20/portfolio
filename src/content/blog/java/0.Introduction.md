# Why java was created
###### There was C, C++ and other languages in the market what problem does java solved
###### C and C++ being --> Fast, simple, low-level, less abstraction
###### java was made as open source by sun micro-systems, it solves problems such as portability(need a compiler compiles for a platform)

> [!Main Idea]
> Java solved : write once and run anywhere (WORA)
### Platform (Portable)
###### Processor + OS combination is a unique platform P-1(x86+windows) P-2(ARM+mac)
###### C and C++ is not cross-platform because ==> as different machine need different machine(similarly different kernel so different system call(like print, input, file read-write, memory allocate)==> different system libraries) and different processor(has logic gates(transistors) which have different arrangement in different processor so interaction with processor is different)==> ISA(instruction set architecture) is different of different processor and ISA is used by kernel to make it's system libraries

> [!Note]
> kernel can only access ISA(hardware level command for processor) and it provides system libraries for the programs
#### BYTE-CODE and JVM
###### It solves the problem of platform to make it cross-platform
###### Like compiler has to learn the language of the platform to run the program ==> it leads to platform dependencies, solution is like by giving it a translator(so it doesn't have to learn the platform)
###### Here the translator is JVM(java virtual machine) is a software -->the compiler will convert to BYTE-CODE(intermediate code) ==>JVM can run this BYTE-CODE ![[Pasted image 20260508213503.png|538]] JVM is present on the platform(there is different JVM for each platform).

> [!Note]
> JVM is platform dependent
#### Why platform-Independence matters
###### So can run a code in many embedded system --> general purpose ==> any server(PC) which should be able to run app
## Simplicity

###### As pointers, multi-inheritance and memory allocation ==> complex which is simplified by Java by removing them.

###### JVM --> solves Security and potability
###### Simplicity --> solved by syntax of Java
