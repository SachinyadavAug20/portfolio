## Multi-dimension array
###### more than 1 dimension [][][] -> 2D array. in memory stores in a line (not matrix)
```java
int [][] marks = new int[10][2];  // roll:marks of 10 students
```

###### Random access in array --> because memory allocation is continuous.same way as in C++--> by using pointer arithematic
```cpp
int arr[5]={1,2,3,4,5};
cout<<arr[3]<<endl;    // 4
cout<<*(arr+3)<<endl;  // 4
cout<<3[arr]<<endl;    // 4
cout<<*(3+arr)<<endl;  // 4
```
###### it is stored in heap(as it is using making `new` keyword). here arr doesn't hold array it reference it's address in heap. arr is base address stored in stack of heap.
###### similar random access in array of array(2D) -> make base address in heap to point at heap
###### For boolean in java, -> boolean has no fixed size(platform dependent as CPU operations are size dependent) it default takes 1 bit. so it's array is made as per it's size there.
