> [!Note]
> Every number is signed(+,-) in java. Thus for n-bits can go from -2^(n-1) to 2^(n-1)-1. The last bit is negative bit
###### Real number => floating point numbers. can represent as float/double range is different
###### float -> single precision *less precise*
###### double -> double precision *slightly more precise* 
###### float not used in production as loss of value (use double more)
###### Java inbuilt function like sin,cos,average etc are double returner
###### character ==> can represent any uni-code(0-65535 and 16-bit) character (other language C/C++ used ASCII(subset of uni-code 0-127 and 8-bit)) 

> [!NOTE]
> In java `''` single quotes for character and `""` double quotes for string literals
###### boolean is either true or false is 1-bit 

> [!Difference]
> In C/C++ any non-zero number is true, but in java this is not the case boolean is only true/false nothing else

```Java
public class main{ // this main is the name of file main.java
  public static void main(String[] args){
    // integers all in base 10
    byte b = 5;
    short s = 22;
    int i = 320;
    long l = 420198;
    System.out.println("Integers: " + b + " " + s + " " + i + " " + l);

    // reals
    float f = 3.2f; // as 3.2 is default double make it float by "3.2f"
    double d = 4.2;
    System.out.println("Reals: " + f + " " + d);

    // characters
    char c = 'a';
    char c2 = 99; // will be c
    // char c3="j"; // error
    System.out.println("Characters: " + c +' '+c2);

    // booleans
    boolean t = true;
    boolean fa = false;
    // boolean boo=1;  // error
    System.out.println("Booleans: " + t + " " + fa);

    // binary(base 2) , octal(base 8), hexa(base 16) system
    // 5 = 101 in binary, 05 in octal, 5 in hexa
    int bin=0b101; // binary starts with 0b
    int oct=005; // octal starts with 0
    int hex=0x5; // hexa starts with 0x
    long bbin=0x1f3;
    System.out.println("Binary,octal,hexa: " + bin + " " + oct + " " + hex+ " "+bbin);

    // real precision
    float f1=3.14f;
    double d1=6.022e23;
    System.out.println("Real precision: " + f1 + " " + d1);

    // readable
    long big=5_000_000_000L; // _ in number is for readable L at the end
    System.out.println(big);
  }
}
```
